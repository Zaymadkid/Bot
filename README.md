# LotBlox Bridge - Discord Bot

## About
LotBlox Bridge is a Discord bot that integrates with the LotBlox browser extension to provide Discord notifications and features.

## Add Bot to Your Server
Click the link below to add the LotBlox Bridge bot to your Discord server:

**[Add LotBlox Bridge Bot](https://discord.com/oauth2/authorize?client_id=1458782837435138194&permissions=0&integration_type=0&scope=bot)**

## Setup
1. Click the invite link above
2. Select the Discord server you want to add the bot to
3. Authorize the bot
4. The bot will now be active in your server!

## Features
- Discord integration with LotBlox browser extension
- Real-time notifications
- Server management features
